Instalation and set up step by step

For the first few settings, I conncted my server with keyboard and screen.

  • Install Debian on server machine
  • Connect to my wireless internet
  • Install openssh-server on server (Debian) and openssh-client on Ubuntu
  • With $ ip a command I found my server ip address
  • On Ubuntu computer, I connected to server with: $ ssh -p < port_number > username@hostname

I was successfuly connected to my server and from now, further steps were done from Ubuntu computer (client).

  • With $ nano /etc/network/interfaces and with help of Debian documentation, I configured static ip address for Wireless connection
  • I have made some security changes in $ nano /etc/ssh/sshd_config
  • Public key Authentication (with ssh_keygen, set passphrase, ssh_copy_id...)

Transfer files and directories using SCP

Files:

$ scp -P < port_num > < source path > username@ip : < target path >

Directories:

$ scp -r -P < port_num > < source path > username@ip : < target path >

Mount server using SSHFS

This is used for easier and faster file transfer.

Install sshfs:

$ sudo apt-get install sshfs

Mount the remote file system:

$ sshfs < target ip >:/yourDir location/on/server

Now "yourDir" is mounted directorie and you can upload files and folders there.
