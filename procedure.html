DIY Server
Hardware Software Procedure

For first few set ups I conncted my server with keyboard and screen.

  • Install Debian on "server" machine
  • Connect to my wireless internet
  • Install openssh-server on server (Debian) and openssh-client on Ubuntu
  • With $ ip a command I found my server ip address
  • On Ubuntu computer, I connected to server with: $ ssh -p < port_number > username@hostname

I was successfuly connected to my server and from now, further steps were done from Ubuntu computer (client).
